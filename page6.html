Threat Detective

Pre-trained image classifier deployed on Google Cloud


An image classification website that will tell you if the image you have uploaded is a firearm, knife or a bomb.
Pretrained resnet34 from FastAI used to build classifier by training on custom dataset of three classes.
The dataset was built from scratch using Google Images and using a JavaScript command that downloaded the URLs into a CSV.
FastAI's download functionality was used to get individual jpeg files into labeled folders.
